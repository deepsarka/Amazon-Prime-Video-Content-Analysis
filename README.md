### 📌 Amazon Prime Video — Content Analysis
An EDA project analyzing Amazon Prime Video's US content library using two datasets: `titles.csv` (9,871 records) and `credits.csv` (124,235 records).

## Objective
Understand content diversity, audience preferences, regional trends, and factors influencing content success on the platform.

## Tools Used: Python, Pandas, NumPy, Matplotlib, Seaborn , Google Colab

## Datasets
| File | Records | Key Columns |
|---|---|---|
| `titles.csv` | 9,871 | title, type, genres, release_year, runtime, imdb_score, tmdb_popularity |
| `credits.csv` | 124,235 | name, role (ACTOR/DIRECTOR), character, id |

## Key Findings

- Movies make up **86%** of the library; TV shows only 14% — yet shows consistently rate **1–1.5 pts higher** on IMDb
- Average IMDb score is **~6.0/10** and has stayed flat across all eras since 1970
- **US dominates** with 4,810 titles (~60%); India is second with 1,048
- Top directors average **7.5–8.5 IMDb** across multiple titles — track record is a reliable quality predictor
- No strong correlation between rating and popularity — marketing matters as much as quality
